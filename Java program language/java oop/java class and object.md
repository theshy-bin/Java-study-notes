# Java类与对象
类：一种自己定义的引用数据类型
对象：类的实例，即引用类型变量
## 对象的组成
类是及具体的属性与行为的提取，对象就是有属性[成员变量/字段]和行为[方法]
## 创建对象以及对象在内存中的形式
对象是一种引用类型变量，即用地址指向存储数据的空间
```java
Cat cat[对象名/对象引用] = new Cat();[创建对象空间（数据）]
```
